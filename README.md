# Modern Furniture & Lifestyle Website
<img width="1919" height="912" alt="image" src="https://github.com/user-attachments/assets/b910a14b-bed6-49bf-bc5f-45d5c88705ef" />

A well  animated lifestyle and furniture showcase website built using **HTML, CSS, JavaScript, and GSAP**. This project features smooth scroll-based animations, modern UI design, and an immersive browsing experience.

## Features

- Smooth scroll animations with GSAP & ScrollTrigger  
- Netflix-style intro text animation  
- Responsive and minimal layout  
- Dynamic theme switching  
- Interactive furniture grid section  
- Clean and modern UI design  

##  Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- GSAP (GreenSock Animation Platform)  

## Purpose

This project was created as a learning and portfolio project to explore modern web animations, UI interactions, and front-end development best practices.
